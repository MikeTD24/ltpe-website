## 🚀 Prochaines Étapes

### ÉTAPE 1 : Configuration API YouTube (CRITIQUE)
1. Créer un projet sur Google Cloud Console
2. Activer l'API YouTube Data V3
3. Générer une clé API
4. Remplacer dans `js/youtube-api.js` :
   - `YOUTUBE_API_KEY = 'YOUR_API_KEY_HERE'`
   - `YOUTUBE_CHANNEL_ID = 'UCltpevangile'`

### ÉTAPE 2 : Ajouter vos images
1. Placer les photos dans le dossier `images/`
2. Remplacer les chemins dans les fichiers HTML

### ÉTAPE 3 : Tester localement
```bash
cd /home/ubuntu/ltpe-website
python -m http.server 8000
# Ouvrir http://localhost:8000
```

### ÉTAPE 4 : Publier sur GitHub
1. Créer un repository sur GitHub
2. Initialiser Git localement
3. Pousser le code
4. Activer GitHub Pages

### ÉTAPE 5 : Déployer en ligne
1. Utiliser Netlify, Vercel ou votre hébergeur
